- Ver.2.1:「月」に”Month”を追加。「玉」の読みに「ぎょく」を追加。
- Ver.2.2: 漢字に英語の曜日を追加。
- Ver.2.3: カタカナの「ヲ」の書き順を由緒正しい方に修正。Windows 2000とWindows 11を正式サポート。EXEファイルにデジタル署名を追加。
- Ver.2.4: メインアイコンを更新。漢字の英語を修正。最小化できるようにした。辞書を更新。
- Ver.2.5: 「wo (o)」「nn (n/m)」など修正。mojiben4に最小化ボタンを追加。
- Ver.2.6: 日本語フォント（梅ゴシック）を追加。「ha (wa)」を修正。画面が小さすぎるときにエラーメッセージを表示した。ローカライズを改良。
- Ver.2.7: 「tsu (tz)」を修正。URLとインストーラを更新。「梅ゴシック」フォントを「ピザPゴシック」で置き換え。日本語フォントをオプションに。ReactOSをサポート。
- Ver.2.8: 数え方を改良。音読みと訓読みを区別。
- Ver.2.9: 小2の漢字を追加。声を機械音声に変更。Escキーで筆を停止できるようにした。高速モードを導入。メモリリークの問題を解決。メインアイコンを一部変更。
- Ver.3.0: 小3の漢字を追加。「数字と数」を改良。カーソル形状による反応を追加。英語のスペルを修正。
- Ver.3.1: 「数字と数」のボタンを修正。「数字と数」に「九九のうた」ボタンを追加。プロジェクトを小学校低学年と小学校高学年に分ける。ABCのメインアイコンを変更。漢字の意味を日本語ローカライズ。
- Ver.3.2: 「Study mode」を導入。英語で学ぶか、日本語で学ぶかを選べるようになった。書き順のリージョンデータを圧縮。 文字の表示・折り返しなどを改良。
- Ver.3.3: 「もじのコピー」を追加。漢字に「Google けんさく」と「Google けんさく (じゅくご)」を追加。漢字に「Jisho.org でしらべる」を追加。インストーラのショートカット ファイルの名前を単純化。漢字の説明をいくつか修正。
- Ver.3.4: 小4の漢字を追加。BMPをGIFに変換してバイナリサイズを削減。
- Ver.3.5: 小5の漢字を追加。Polarストロークの表示を修正。
- Ver.3.6: 小6の漢字を追加。漢字の書き順ウィンドウのサイズを300×300から254×254に縮小。漢字の日本語の説明を修正。
- Ver.3.7: 漢字の書き順のリージョンデータをリマスタリング。カナとアルファベットの書き順ウィンドウを254x254に縮小。フリガナを実装。

# Change log of "Moji No Benkyou"

- Ver.2.6: Added Japanese font (Ume Gothic). Fixed “ha (wa)”. Displayed error message when screen is too small. Improved localization.
- Ver.2.7: Fixed “tsu (tz)”. Updated URLs and installer. Replaced “Ume Gothic” font with “Piza P Gothic”. Made Japanese font optional. Supported ReactOS.
- Ver.2.8: Improved numbering. Distinguished between on-yomi and kun-yomi.
- Ver.2.9: Added 2nd-grade kanji for elementary school. Changed the voices to machine voices. Added ability to stop the strokes with Esc key. Introduced high-speed mode. Fixed the memory leak issue. Partially changed the main icons.
- Ver.3.0: Added kanji for third graders. Improved “Digits and Numbers.” Added cursor shape responses. Corrected English spelling.
- Ver.3.1: Fixed buttons of “Digits and numbers”. Added the “9×9 song” button on “Digits and numbers”. Split the project to “Elementary Low” and “Elementary High”. Changed the main icon of ABC.
- Ver.3.2: Introduced “Study mode.” You can now choose to study in English or Japanese. Compressed stroke order region data. Improved character display and wrapping.
- Ver.3.3: Added “Copy character”. Added “Google search” and “Google search (Idioms)” to Kanji. Added “Look up Jisho.org” to Kanji. Simplified installer shortcut filenames. Fixed some Kanji descriptions.
- Ver.3.4: Added kanji for 4th grade. Converted BMP to GIF to reduce binary size.
- Ver.3.5: Added kanji for 5th grade. Fixed Polar strokes display.
- Ver.3.6: Added kanji for sixth grade. Reduced the size of the kanji stroke order window from 300×300 to 254×254. Fixed the Japanese explanations for kanji.
- Ver.3.7: Remastered regional data for kanji stroke order. The kana and alphabet stroke order window has been reduced to 254x254. Furigana is implemented.
