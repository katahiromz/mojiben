もじのべんきょう
   by 片山博文MZ

☆概要☆

「もじのべんきょう」は、文字の発音と書き順の学習を補助する画期的なソフトです。

「ひらがな」「カタカナ」「アルファベット」「小１～小６のかん字」を学習できます。


☆使い方☆

日本語フォント「Piza-P-Gothic.ttf」が付属している場合は、事前にそれをインストールしてください。

文字のボタンを押すと、その文字の音声が再生され、書き順が表示されます。

☆動作環境☆

	* Windows 2000
	* Windows XP
	* Windows Vista
	* Windows Server 2003
	* Windows 10
	* Windows 11

☆アンインストール方法☆

インストーラ付きの場合は、コントロールパネルの
「アプリケーションの追加と削除」 （プログラムの追加と削除）から
アンインストールできます。

インストーラなしの場合は、ファイルを削除するだけです。


☆お知らせ☆

このソフトウェアは、日本語と英語の2言語に対応しています。
別の言語で楽しみたい場合は、タイトルバーのシステムメニューから "Study mode" をお選びください。

☆使用許諾☆

「もじのべんきょう」は、フリーソフトです。
転載・配布は自由に行ってもかまいません。
このプログラムを使って生じた如何なる損害も当方は責任を負いません。

☆感謝(かんしゃ)の言葉(ことば)☆

このプログラムをお使(つか)いくださりありがとうございます。

文字(もじ)には音(おと)を表す文字と意味(いみ)を表(あらわ)す文字があります。
ひらがなとカタカナは音を表す文字です。漢字(かんじ)は1個(こ)以上の音と意味を持(も)ちます。
さらに複数(ふくすう)の漢字が組(く)み合(あ)わさると熟語(じゅくご)になり、別(べつ)の意味が生まれます。
文字は読(よ)み書(か)きと発声(はっせい)と理解(りかい)ができてはじめて習得(しゅうとく)したと言(い)えます。

正(ただ)しい書き順(じゅん)は文字をきれいに書(か)いたり、物事(ものごと)をテキパキ片付(かたづ)ける訓練(くんれん)になります。書き順を軽視(けいし)しないでください。

漢字には音読(おんよ)みと訓読(くんよ)みの二種類(にしゅるい)の読(よ)み方(かた)があります。
音読みは古代(こだい)中国(ちゅうごく)から伝(つた)わった読み方です。訓読みは日本(にほん)独自(どくじ)の読み方です。

最初(さいしょ)はひらがなとカタカナから勉強(べんきょう)すると良(よ)いでしょう。
