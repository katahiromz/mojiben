もじのべんきょう (3)
   by 片山博文MZ
--------------------

☆概要☆

「もじのべんきょう (3)」は、 小学校一年生で習う漢字の
発音と書き順の学習を補助する画期的なソフトです。


☆使い方☆

漢字のボタンを押すと、その漢字の音声が再生され、書き順が表示されます。


☆動作環境☆

	* Windows XP
	* Windows Vista
	* Windows Server 2003
	* Windows 10

☆アンインストール方法☆

インストーラ付きの場合は、コントロールパネルの
「アプリケーションの追加と削除」 （プログラムの追加と削除）から
アンインストールできます。

インストーラなしの場合は、ファイルを削除するだけです。


☆使用許諾☆

「もじのべんきょう (3)」は、フリーソフトです。
転載・配布は自由に行ってもかまいません。
このプログラムを使って生じた如何なる損害も当方は責任を負いません。
