もじのべんきょう (1)
   by 片山博文MZ
--------------------

☆概要☆

「もじのべんきょう (1)」は、 ひらがなとカタカナの文字の
発音と書き順の学習を補助する画期的なソフトです。


☆使い方☆

文字のボタンを押すと、その文字の音声が再生され、書き順が表示されます。
上部のボタンで、ひらがなとカタカナを切り替えることができます。


☆動作環境☆

	* Windows XP
	* Windows Vista
	* Windows Server 2003
	* Windows 10

☆アンインストール方法☆

インストーラ付きの場合は、コントロールパネルの
「アプリケーションの追加と削除」 （プログラムの追加と削除）から
アンインストールできます。

インストーラなしの場合は、ファイルを削除するだけです。


☆使用許諾☆

「もじのべんきょう (1)」は、フリーソフトです。
転載・配布は自由に行ってもかまいません。
このプログラムを使って生じた如何なる損害も当方は責任を負いません。
