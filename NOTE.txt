# 備忘録

音声生成はNoahさんのWindows用フリーウェア「Text To Wav」の「Microsoft Ichiro - Japanese」を使用する。

音声編集はCoderiumさんの「Sound Engine」を使用する。

MP3エンコード・デコードは海猫さんの「午後のこ～だ」を使用する。

フォント「モトヤ教科書3」は、栃木の「栃」を間違えている（ウソ字）。

このプログラムは、英語版の Windows 2000でも動作するように設計しないといけない。
そのため、フォントの文字セットの設定を間違ってはならない。

`kkjplay.exe`は、書き順プレーヤー。KKJファイルをドロップして Playボタンを押すと再生する。
ただし、付属のRGNファイルが必要。

`bmp2rgn.exe` は BMPファイルをリージョンデータ（RGNファイル）に変換するプログラム。

`rgn2bmp.exe` は RGNファイルをBMPファイルに変換するプログラム。

`makebmp.exe` は指定した文字のBMPファイルを作成するプログラム。

RGNファイルとは、254x254ピクセルの中の黒いピクセルの領域を定義するファイル。
RGNファイルの定義については `Region.cpp` の `DeserializeRegion254`関数と`SerializeRegion254`関数を参照。
